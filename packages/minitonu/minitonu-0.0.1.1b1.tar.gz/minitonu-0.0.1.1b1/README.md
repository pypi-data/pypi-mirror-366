# Minitonu

Minitonu es una microbiblioteca inspirada en PyTorch y NumPy, escrita en puro Python sin dependencias externas.

## Características
- Tensores simples
- Perceptrón básico
- Pruebas integradas
- Ideal para uso en Termux o sistemas sin pip

## Uso rápido

```bash
pip install minitonu
python -m minitonu
