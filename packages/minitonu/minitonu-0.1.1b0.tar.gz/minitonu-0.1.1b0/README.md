minitonu

**minitonu** es una microbiblioteca experimental sin dependencias, inspirada en `torch` y `numpy`.

- Compatible con Pydroid3, Termux y otros entornos Python puros.
- Incluye operaciones básicas tensoriales y conexiones de red neuronal simplificadas.
- No requiere instalación de dependencias externas.
- Ideal para pruebas educativas, CLI o entornos limitados.

## Instalación

```bash
pip install minitonu --upgrade
